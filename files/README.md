# SDT104 Homework 11

This project is a small JavaScript DOM assignment for SDT104.

It renders a list of movie cards dynamically from an array of objects in `cards.js`.

Features:
- dynamic card creation with DOM methods
- click event handling for card selection
- search filtering
- alphabetical sorting
- array method examples with `reduce()`, `sort()`, `filter()`, and `map()`

Files:
- `index.html` - page structure and styles
- `cards.js` - data objects, rendering logic, event handling, and array method examples

To open the project, load `index.html` in a browser.
